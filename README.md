# Anmeldte lovbrudd i norske kommuner

Denne siden ble laget for Samarbeidsdesken og lastet opp første gang 19.12.2025.

Den inneholder tall fra politiet / SSB over anmeldte lovbrudd, brutt ned på kategorier kriminalitet og kommuner.

Tallene er vasket av Christian Lura, Samarbeidsdesken og deles med deskens samarbeidsredaksjoner.
